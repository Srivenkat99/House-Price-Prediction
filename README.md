# 🏠 House Price Prediction using Machine Learning

A Machine Learning web application built with **Flask** that predicts California house prices based on user input.

---

## 📌 Project Overview

This project uses the California Housing dataset from Scikit-Learn to predict house prices using a trained Machine Learning model.

Users can enter housing details through a web interface, and the application predicts the estimated house price instantly.

## 📝 Project Notes

- This is my **first end-to-end Machine Learning project**.
- The primary goal of this project was to practice and understand the complete ML workflow, including **Regression, Flask, Git, and GitHub**.
- As this was my first project, I used **ChatGPT as a learning guide** to understand concepts, debug issues, and improve the implementation. I wrote, tested, and modified the project while learning each step.
- This is a **baseline implementation** intended for learning purposes. Therefore, advanced techniques such as:
  - Exploratory Data Analysis (EDA)
  - Feature Engineering
  - Feature Scaling
  - Pipelines
  - Hyperparameter Tuning
  - Cross Validation
  were intentionally not included.
- I plan to improve future versions of this project by incorporating these techniques as I continue learning Machine Learning.

## ⚙️ Installation

Clone the repository:

```bash
git clone <repository-url>
```

Move into the project directory:

```bash
cd House_Price_Prediction
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
